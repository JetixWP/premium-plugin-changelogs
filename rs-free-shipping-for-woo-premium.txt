Free Shipping for WooCommerce products extends default free shipping workflow of WooCommerce to conditionally apply free shipping on multiple products and more.

== Description ==

== Installation ==

This section describes how to install the plugin and get it working. e.g.

1. Upload the plugin files to the `/wp-content/plugins/rs-free-shipping-for-woo-premium` directory, or install the plugin through the Plugins >> Add new screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Head to 'Free Shipping for WooCommerce' page from WordPress sidebar menu.
